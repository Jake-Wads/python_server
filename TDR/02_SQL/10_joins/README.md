# Joins
https://ds.codeup.com/sql/joins/

## Lesson Objectives
Students will learn:
- what it means to join tables
- primary and foreign keys
- how to identify row relationships types
    - one to many
    - many to many
- basic different types of joins
    - inner
    - outer
    - left
    - right
- how to do multiple joins
