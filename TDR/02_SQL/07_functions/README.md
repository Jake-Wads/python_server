# Functions
https://ds.codeup.com/sql/functions

## Lesson Objectives
Students will learn: 

- To use the following type of functions
    - String types
        - CONCAT
        - SUBSTR
        - UPPER / LOWER
        - REPLACE
    - Date types
        - NOW
        - CURDATE 
        - CURTIME 
    - Numeric types
        - AVG
        - MIN
        - MAX
- how to CAST
