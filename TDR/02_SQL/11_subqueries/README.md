# Subqueries
https://ds.codeup.com/sql/subqueries/


## Lesson Objectives
Students will learn:
- the difference between a subquery and a join
- how to use subqueries
    - scaler subqueries
    - column subqueries
    - row subqueries
    - table subqueries
