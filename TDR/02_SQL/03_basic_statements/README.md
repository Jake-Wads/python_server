# Basic Statements
https://ds.codeup.com/sql/basic-statements/

## Lesson Objectives
Students will learn basic SQL syntax including:
- Query statements
    - SELECT
    - FROM 
    - DISTINCT
    - WHERE
- Comments
- Quotes
- Operators
- Aliasing
- Whitespace
